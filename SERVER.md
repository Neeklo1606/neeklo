# Сервер и каталог проекта

## Продакшен (BeGet)

| Параметр | Значение |
|----------|----------|
| Хостинг | BeGet |
| SSH-хост | `dragon.beget.ru` |
| Пользователь SSH | `dsc23ytp` |
| Подключение | `ssh dsc23ytp@dragon.beget.ru` |
| Каталог проекта на сервере | `~/neeklo/public_html` |
| Полный путь (типично для BeGet) | `/home/d/dsc23ytp/neeklo/public_html` |

Сайт в продакшене: **https://neeklo.ru** (и зеркала по настройкам панели BeGet).

Документ веб-сервера для Laravel должен указывать на каталог **`public`** внутри этой директории (`.../public_html/public`), если так настроено в панели.

Пароли и ключи в этот файл не заносятся — только в `.env` на сервере и в менеджере секретов.
